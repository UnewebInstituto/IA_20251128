## üíª Ejemplo Pr√°ctico: Generador de Ideas de Contenido con Gemini

Crearemos una aplicaci√≥n simple donde el usuario ingresa un tema (en el HTML) y Node.js usa Gemini para generar ideas.

### **Paso 1: Configuraci√≥n del Proyecto (Node.js - Backend)**

Necesitas tener Node.js instalado.

1.  **Crea el directorio del proyecto:**

    ```bash
    mkdir gemini-web-app
    cd gemini-web-app
    npm init -y
    ```

2.  **Instala las dependencias necesarias:**

      * **`@ibm/gen-ai`:** El SDK de IBM para interactuar con Gemini (asumo que te refieres al SDK de IBM, ya que usaste `pip install genai` anteriormente, aunque la API de Google Gemini usa `@google/genai`).
      * **`express`:** Para crear el servidor API REST.
      * **`cors`:** Para permitir que el *frontend* (HTML) acceda al *backend*.
      * **`dotenv`:** Para manejar la clave API de forma segura.

    <!-- end list -->

    ```bash
    npm install @ibm/gen-ai express cors dotenv
    ```

3.  **Crea el archivo `.env` (Configuraci√≥n de Seguridad):**

      * En la ra√≠z del proyecto, crea un archivo llamado `.env` y pega tu clave API. **Nunca expongas esta clave en el c√≥digo frontend.**

    <!-- end list -->

    ```dotenv
    # Reemplaza 'TU_CLAVE_API_AQUI' con tu clave real.
    GEMINI_API_KEY="TU_CLAVE_API_AQUI" 
    ```

4.  **Crea el archivo `server.js` (API Backend):**

Este archivo manejar√° la l√≥gica de la API de Gemini.

```javascript
// server.js (Backend Node.js)

require('dotenv').config();
const express = require('express');
const cors = require('cors');
const { Client, Models } = require('@ibm/gen-ai'); // Usando el SDK de IBM genai

// --- Configuraci√≥n ---
const app = express();
const port = 3000;

// Inicializaci√≥n del cliente IBM Gen AI con la clave del .env
const client = new Client({ apiKey: process.env.GEMINI_API_KEY });
const modelId = Models.GEMINI_20B_CHAT; // Modelo recomendado para tareas de chat/texto

// Middlewares
app.use(cors()); // Permite peticiones desde el frontend (puerto 8080, por ejemplo)
app.use(express.json()); // Permite analizar el cuerpo de las peticiones JSON

// --- Endpoint de la API ---
app.post('/generate-ideas', async (req, res) => {
    const { topic } = req.body;

    if (!topic) {
        return res.status(400).json({ error: 'El campo "topic" es requerido.' });
    }

    // 1. Crear el prompt (instrucci√≥n para Gemini)
    const prompt = `Act√∫a como experto en marketing digital. Genera 5 ideas de contenido creativas y detalladas sobre el siguiente tema: "${topic}". Las ideas deben ser adecuadas para un blog.`;

    try {
        // 2. Llamada a la API de Gemini
        const response = await client.text.generate({
            modelId: modelId,
            input: prompt,
            parameters: {
                decodingMethod: 'greedy',
                maxNewTokens: 500,
            }
        });

        // 3. Devolver la respuesta al frontend
        const generatedText = response.results[0].generatedText;
        res.json({ success: true, ideas: generatedText });

    } catch (error) {
        console.error('Error al llamar a la API de Gemini:', error);
        res.status(500).json({ success: false, error: 'Error interno del servidor al generar contenido.' });
    }
});

// Iniciar el servidor
app.listen(port, () => {
    console.log(`Servidor Node.js corriendo en http://localhost:${port}`);
    console.log(`¬°Listo para recibir peticiones POST a http://localhost:${port}/generate-ideas!`);
});
```
